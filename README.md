# CSCI 570 Sequence Alignment 
Implementation of the *Sequence Alignment* problem using 2 different methods: Dynamic Programming (DP), and Memory-efficient version (DP + Divide & Conquer)

# Running the Program
To run the DP version:
'python3 basic.py input.txt output.txt'

To run the Memory-efficient version:
'python3 efficient.py input.txt output.txt'
